Z Object Publishing Environment

The Z Object Publishing Environment, aka Zope, is a production of Digital Creations. Many individuals have contributed code and ideas to this project. We would not be where we are today with such strong community support. Jeff Bauer in particular deserves recognition as the leader of the PCGI project. Michael Hauser came up with the name Zope.

Thanks also to Guido van Rossum and the Python community for the wonderful Python language.

To find our more about Zope visit the Zope web site.
